## 1. Get Your API Key

Sign up at [humandesignapi.nl](https://humandesignapi.nl) and choose a plan. Your API key will be available in the [dashboard](https://humandesignapi.nl/dashboard).

## 2. Get a Google Geocoding Key

All endpoints require a Google Geocoding API key to convert place names into coordinates. You need a [Google Geocoding API key](https://developers.google.com/maps/documentation/geocoding/get-api-key).

## 3. Make Your First Request

<CodeGroup>
```bash cURL
curl -X POST https://api.humandesignapi.nl/v1/bodygraphs/simple \
  -H "Content-Type: application/json" \
  -H "HD-Api-Key: YOUR_API_KEY" \
  -H "HD-Geocode-Key: YOUR_GEOCODE_KEY" \
  -d '{
    "birthdate": "15-Jan-90",
    "birthtime": "14:30",
    "location": "Amsterdam, The Netherlands"
  }'
```

```javascript JavaScript
const response = await fetch(
  "https://api.humandesignapi.nl/v1/bodygraphs/simple",
  {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "HD-Api-Key": "YOUR_API_KEY",
      "HD-Geocode-Key": "YOUR_GEOCODE_KEY",
    },
    body: JSON.stringify({
      birthdate: "15-Jan-90",
      birthtime: "14:30",
      location: "Amsterdam, The Netherlands",
    }),
  },
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    "https://api.humandesignapi.nl/v1/bodygraphs/simple",
    headers={
        "Content-Type": "application/json",
        "HD-Api-Key": "YOUR_API_KEY",
        "HD-Geocode-Key": "YOUR_GEOCODE_KEY",
    },
    json={
        "birthdate": "15-Jan-90",
        "birthtime": "14:30",
        "location": "Amsterdam, The Netherlands",
    },
)

data = response.json()
print(data)
```

</CodeGroup>

## 4. Understand the Response

The API returns chart data directly as a JSON object with `snake_case` field names:

```json
{
  "type": "Generator",
  "profile": "6/2",
  "gates": ["20", "34", "10", "57"],
  "channels_short": ["20-34", "10-57"],
  "centers": ["G", "Sacral", "Spleen", "Throat"]
}
```

| Field            | Description                                        |
| ---------------- | -------------------------------------------------- |
| `type`           | The Human Design type (e.g., Generator, Projector) |
| `profile`        | The profile (e.g., 6/2)                            |
| `gates`          | List of active gates                               |
| `channels_short` | List of defined channels in short format           |
| `centers`        | List of defined centers                            |
